# HeadyConnection Foundation

**"The Why" - Non-Profit Entity**

HeadyConnection Foundation defines the philosophical and architectural foundation of the Heady Directive. It establishes the interfaces, schemas, and policies for the sovereign AI ecosystem.

## Purpose

HeadyConnection acts as the "Creative Child" - establishing:
- Identity Layer (DID interfaces, Identity Schema, Sovereign Policy)
- Interface Layer (Core & Protocol interfaces)
- Schema Layer (Provisionals & Governance schemas)
- Policy Layer (Non-profit & Interface policies)

## Structure

```
heady-connection/
├── src/
│   ├── identity/       # DID interfaces and identity schemas
│   ├── interfaces/     # Core and protocol interfaces
│   ├── schemas/        # Provisionals and governance schemas
│   └── policies/       # Non-profit and interface policies
├── _AI_CONTEXT/        # Immutable Provisionals and Master Prompt
└── package.json
```

## Relationship to HeadySystems

- **HeadyConnection**: Defines "The Why" (philosophy, interfaces, schemas)
- **HeadySystems**: Implements "The How" (hardened implementation, C-Corp)
