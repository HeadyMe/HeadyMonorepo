### New Command: `hc -a deploy`

**Purpose:** Automated stage â†’ commit â†’ verify â†’ push â†’ sync

**What It Does:**
1. Stage all changes (`git add -A`)
2. Commit with message (`git commit`)
3. Verify with checkpoint (`hs -Checkpoint`)
4. Push to remotes (`git push`)
5. Run HeadySync (`hs`)

**Usage:**
```bash
# Full automated deployment
hc -a deploy

# With custom commit message
hc -a deploy -m "feat: your changes"

# Force deployment (skip validation)
hc -a deploy -Force
```
